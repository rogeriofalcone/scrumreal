about site

This site is a demo with some tools for you to use during the implementation of Scrum. Unlike other online tools for Scrum, the goal is merely to facilitate the generation of postits with tasks, burndown charts, planning poker.

The initial idea was to learn to use the Flask, so I decided to implement this little project to motivate studies. The site is based on an original idea of Rudy Matela. He made similar tool using TeX
